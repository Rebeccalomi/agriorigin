鸡蛋区块链溯源后台管理系统
项目介绍
"AgriOrigin"--基于区块链的鸡蛋溯源平台采用一盒一码的方式实现对农产品的溯源，运用区块链技术对二维码信息进行保护，防止其信息被更改，最终实现消费者通过扫描商品盒上的二维码就能获取该农产品的详细信息，让消费者放心使用。我们通过与企业进行合作，为企业、消费者提供服务，实现盈利和后续开发。 我们以将企业提供的每盒鸡蛋作为单独溯源对象，实现从生产到销售全部环节的数据记录，生成由区块链溯源技术支持的定制防伪二维码，并粘贴到合格的产品上，提供给消费者查询。


文件架构
该文件夹的功能为
api:guns框架自带java项目文件
system:包含了controller,dao,model的javaweb主要核心文件
web3j:团队导入的web3j--以太坊区块链API组件
